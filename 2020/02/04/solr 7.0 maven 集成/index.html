项目背景：

最近工作用到solr做搜索，觉得挺有趣，自己创建一套引擎，也方便研究。
看网上基本都是手动配置的，拷贝jar包，没有maven集成的，有集成solr版本也很老。
所以先自己与maven 集成了一下。
之后 可能会有一些 定制化的功能添加到这个项目中。如自定义分词器，自定义评分等。

solr使用：
